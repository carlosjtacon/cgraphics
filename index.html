Development and Programming Projects

I have worked on multiple projects both personal and related to university. You can check out my personal GitHub page where some projects' source code is available for free.

Some featured projects:

  • Morfilms: I made this app to keep track of films I wanted to watch when released in theaters. I ended up publishing it in Android - Google Play and iOS - App Store. Source code can be accessed in this repo. I developed morfilms using web technology: Angular and Ionic Framework.
  • Showtimes API: Before making morfilms I was already interested on ways to handle cinema and movies related data in order to make mobile apps. I made an API to store and offer showtimes info by city, location services for cinemas and also information about movies, getting information from various providers. It's deprecated now, but still available. Made with Java/Spring.
  • University Projects: I also have university activities and projects published, some of them are interesting, for example I developed some known algorithms in python to learn new ways of solving problems by programming: divide and conquer, dynamic programming and backtracking.
